Telegram Solo Leveling Fitness Bot

1. Install dependencies:
   npm install node-telegram-bot-api

2. Replace 'YOUR_TELEGRAM_BOT_TOKEN' in index.js with your bot token.

3. Run the bot:
   node index.js

Commands:
/start   - Start your journey
/quests  - Get daily workout quests based on your level
/log     - Log a completed quest (must match today's quest text)
/stats   - See your current XP, level, and progress
/reset   - Reset your progress

Daily quests change once every day and get harder as your level increases.
